# Contributing

Contributions are welcome!

This list is automatically updated via a GitHub Action that pulls data from [Awesome-Bangladeshi-Devs](https://github.com/sharf-shawon/Awesome-Bangladeshi-Devs).

### How it works

1. The script fetches the list of Bangladeshi developers.
2. It checks if each developer has a repository named exactly after their GitHub username (their profile portfolio).
3. If found, it's added to this list.

### To be included

If you are a Bangladeshi developer and want to be on this list:
1. Ensure you are listed in [Awesome-Bangladeshi-Devs](https://github.com/sharf-shawon/Awesome-Bangladeshi-Devs).
2. Create a public repository on GitHub with the same name as your GitHub username.
3. Add a `README.md` to that repository.

### Quality Standards

- The repository must be a "Profile README" (repo name == username).
- It should be well-maintained and provide value to the community.
